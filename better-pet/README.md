# Better Pet（更好的宠物）

适用于 Minecraft 1.21.11 / Fabric 的宠物行为辅助 mod，支持已驯服的猫、狼（狗）和鹦鹉。

## 使用方法

1. 先让自己的宠物坐下；
2. 按住 `Ctrl + Shift`，右键该宠物；
3. 宠物会站起来，并记住脚下方块的坐标；
4. 它会在该坐标水平半径 30 格内自由活动，不再一直跟随主人。

处于范围游荡模式时，宠物项圈下方中央会显示一个小黄点，外观看起来像一颗简化的铃铛。

要恢复普通跟随模式，正常右键让宠物坐下，再正常右键让它站起即可。普通右键站起的宠物会按原版逻辑跟随主人。

## 规则说明

- 只有宠物主人能设置范围游荡模式；
- 只接受近距离交互，并由服务端再次校验，避免伪造请求；
- 中心坐标会随宠物数据保存，退出世界或重启服务端后仍然有效；
- 30 格限制按 X/Z 水平距离计算；
- 若宠物接近边界，其寻路目标会被限制在范围内；被外力推出边界时会被安全地约束回边界内；
- 普通坐下会立即取消范围游荡模式。

## 安装

客户端和服务端都需要安装：

1. `better-pet-1.0.0.jar`；
2. Fabric API `0.141.4+1.21.11` 或兼容版本；
3. Fabric Loader `0.18.4` 或更高版本。

单人游戏中只需安装到当前游戏实例。

## 构建

在仓库根目录运行：

```powershell
.\gradlew.bat :better-pet:build
```

成品位于 `better-pet/build/libs/better-pet-1.0.0.jar`。
